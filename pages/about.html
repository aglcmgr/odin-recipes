About The Project

I've been in love with cooking ever since I was a kid. There's something magical about combining simple ingredients to create something delicious that brings people together. Over the years, I've collected recipes that I absolutely love—dishes that have become favorites in my home and in my heart.

When I started learning web development through The Odin Project, I was excited to discover that they included a recipe website as a mini project! It was the perfect opportunity to combine my interest in cooking with my new web development skills. I collected my favorite recipes to build this practical project, and I hope you enjoy exploring it!

Currently Learning

I'm focused on HTML foundations, semantic HTML, proper structure, elements, and tags. I've also started exploring CSS a bit because I got curious about adding some style to my website lol. It's so fun to see how a little CSS can transform the look of a page, so I've been experimenting with styling as I go.

What I've Learned So Far

I've learned how to use the terminal and command line basics, which feels amazing! I've also understood Git and GitHub—how to initialize repositories, make commits, push to GitHub, and manage version control. These are tools I'm continuously familiarizing myself with because I know they're essential for any developer.

I'm getting more comfortable with VSCode every day! Learning all the keyboard shortcuts, extensions, and features is exciting. There's so much to discover, and I love how my text editor can help me code more efficiently. I still have so much more to learn, but I'm genuinely happy and proud that I was able to build this website on my own! It feels amazing to see my ideas come to life. 💛
