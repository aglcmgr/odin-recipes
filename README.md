# Odin Recipes
## Project Overview
Odin Recipes is a multi-page website I created as part of The Odin Project's Foundations curriculum. This serves as a collection of curated recipes organized into categories such as Appetizers, Chicken, Pork, Beef, Fish, Pasta, and Desserts.

Users can browse recipe categories, view individual recipe pages, learn more about the creator through the About page, leave feedback on the Reviews page, and stay updated through the Blog page. The project was built primarily to practice HTML fundamentals, website structure, navigation, content organization, and basic CSS styling.

This project serves as a foundation for future enhancements as I continue learning CSS, JavaScript, and modern web development practices through The Odin Project.

While the original assignment focused primarily on HTML, I also incorporated basic CSS styling to improve the layout and user experience.

### Skills Practiced
- **Page Structure**: HTML document setup, Head and body sections
- **Text Content**: Headings, Paragraphs, Bold text, Italic text
- **Navigation**: Internal page links, External links, Email links
- **Images**: Image insertion, Alt text
- **Lists**: Unordered lists, List items
- **Layout**: Div containers, Footer
- **CSS**: Classes, Internal styles, Inline styles
- **Multi-Page Website**: Home page, About page, Reviews page, Blog page, Recipe pages
- **Git & GitHub**: Git version control, GitHub repository management
- **Project Planning & Organization**: File structure and project organization

### CSS Skills
Basic color, style, containers
